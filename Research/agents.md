# agents.md — Research (inherits /AGENTS.md)

**Scope:** all research source cards, centralized by topic cluster.

## Rules
- **Cards are thin** (≤ ~80 lines): TL;DR + key verbatim data + citation + retrieval handle (`source_url`) + staleness fields (`as_of`, `revisit_when`, `invalidated_by`).
- **Syntheses do NOT live here.** Route each cluster's synthesis to where it's used — general/wide topic → `Docs/`, project-specific → `Projects/<X>/` — with a backlink to the cluster.
- **Lean access:** read a cluster's `README.md` + its routed synthesis; open cards JIT (search), never bulk-load a cluster.
- Limits: 1 card per source · 1 synthesis per cluster · ≤ ~2 active clusters at a time · archive a cluster once synthesized.

## Process
New research → `deep-research` skill: level gate (L0 ≤3 sources = one dated note; L1/L2 = fan-out) → cards here → synthesis routed out → cluster README updated.
