# Ops/System

The daily machinery. Scheduled tasks, capability modules, backlog, and operating system tracker.

- `modules/` — shared capability logic invoked by skills and scheduled tasks.
- `ScheduledTasks/` — recurring task specs (one file per cadence/trigger).
- `decisions.md` — append-only system-wide decision log.
- `Backlog.md` — running task backlog.
